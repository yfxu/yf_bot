## osu!
Gain a quick glance at your osu! profile in any server to flex your rank \**or to show them how bad you are :)**

## tweet generator
yf_bot can generate a fake tweet given any user's Twitter ID. It will only use words that can be found in the given account's latest tweets. The algorithm uses an unweighted Markov chain to string together words into a semi-comprehensible sentence.
